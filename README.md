        <p>Ночное небо всегда завораживало человечество. Сверкающие звезды, таинственные созвездия – они манили своей красотой и загадочностью. С древних времен люди придумывали мифы и легенды, чтобы объяснить этот величественный спектакль. Сегодня мы совершим путешествие по ночному небу, чтобы познакомиться с некоторыми из самых известных и красивых созвездий: могучим Орионом, неутомимыми Медведицами и прекрасной Андромедой.</p>

        <div class="constellation">
            <h2>Орион: охотник, восставший из пепла мифов</h2>
            <p>Орион – одно из самых узнаваемых созвездий, его легко найти на зимнем небе. Яркие звезды Бетельгейзе (красный сверхгигант) и Ригель (голубой сверхгигант) образуют его плечи и ноги, а три звезды, образующие пояс Ориона (Альнитак, Альнилам и Минтака), служат отличным ориентиром.</p>
<img src="орион.jpg" alt="Орион">

            
            <h3>Мифы и легенды о Орионе:</h3>
            <p class="myth">— Древнегреческая версия: Орион был великим охотником, сыном Посейдона и Эвриалы, сестры Горгон. Он хвастался своей силой и отвагой, за что навлек гнев богини Артемиды. Существует несколько вариантов его смерти: он был убит скорпионом (и тогда созвездие Скорпиона находится на небе напротив Ориона), или Артемида сама убила его, или же он погиб, пытаясь изнасиловать ее. После смерти Орион был вознесен на небо, где продолжил охотиться, но уже среди звезд.</p>
            <p class="myth">— Другие культуры: Мифы об Орионе встречаются в различных культурах. В Египте он ассоциировался с богом Осирисом, а в скандинавской мифологии был известен как охотник, преследующий медведя.</p>
            
            <h3>Помимо мифов:</h3>
            <p>Орион – созвездие, которое поражает не только своей красотой, но и научным значением. В его пределах находится туманность Ориона – одна из самых ярких и известных туманностей, где активно формируются новые звезды.</p>
        </div>

        <div class="constellation">
            <h2>Большая и Малая Медведицы: хранительницы небесного круга</h2>
            <p>Большая Медведица (Ursa Major) – одно из самых больших и известных созвездий, которое легко найти благодаря характерной форме ковша. Малая Медведица (Ursa Minor), в котором находится Полярная звезда (α UMi), менее яркая, но не менее важная.</p>
            <img src="медведицы.jpg" alt="Большая и Малая Медведицы">

            <h3>Мифы и легенды о Медведицах:</h3>
            <p class="myth">— Древнегреческая версия: Согласно мифу, Большая Медведица – это нимфа Каллисто, возлюбленная Зевса. Ревнивая Гера превратила Каллисто в медведицу. Сын Каллисто, Аркад, встретил ее во время охоты и собирался убить, не узнав мать. Тогда Зевс, чтобы спасти их, превратил Аркада в Малую Медведицу, а Каллисто вознес на небо.</p>
            <p class="myth">— Полярная звезда: Полярная звезда, расположенная в хвосте Малой Медведицы, играет ключевую роль в навигации. Она всегда указывает на север, позволяя ориентироваться путешественникам и мореплавателям с древних времен.</p>
        </div>

        <div class="constellation">
            <h2>Андромеда: прекрасная принцесса, прикованная к скале</h2>
            <p>Созвездие Андромеды (Andromeda) находится рядом с созвездием Кассиопеи (мать Андромеды) и Пегаса (крылатый конь). Андромеда известна не только своей красотой, но и наличием галактики Андромеды (M31) – ближайшей к нашей галактике, Млечному Пути.</p>
            <img src="андромеда.jpg" alt="Андромеда">

            <h3>Мифы и легенды об Андромеде:</h3>
            <p class="myth">— Древнегреческая версия: Андромеда была дочерью эфиопского царя Кефея и царицы Кассиопеи. Кассиопея похвасталась, что ее красота превосходит красоту нереид (морских нимф), за что боги наслали на страну морское чудовище Кето. Чтобы умилостивить чудовище, Андромеду приковали к скале. Герой Персей, увидев Андромеду, спас ее, убив Кето, и женился на ней. После смерти Андромеда была вознесена на небо.</p>
            
            <h3>Помимо мифов:</h3>
            <p>Галактика Андромеды (M31) – захватывающее зрелище, которое можно увидеть даже невооруженным глазом в ясную ночь. Она является одним из самых ярких объектов в небе, и ее изучение дает нам ценную информацию о структуре и эволюции галактик.</p>

        document.addEventListener('DOMContentLoaded', function() {
            const likeButton = document.getElementById('likeButton');
            const likeCount = document.getElementById('likeCount');
            const likeText = document.getElementById('likeText');
            const likeMessage = document.getElementById('likeMessage');
            
            let likes = localStorage.getItem('articleLikes');
            let isLiked = localStorage.getItem('isLiked');
            
            if (likes === null) {
                likes = 0;
                localStorage.setItem('articleLikes', likes);
            } else {
                likes = parseInt(likes);
            }
            
            if (isLiked === null) {
                isLiked = 'false';
                localStorage.setItem('isLiked', isLiked);
            }
            
            updateLikeButton();
            
            function updateLikeButton() {
                likeCount.textContent = likes;
                
                if (isLiked === 'true') {
                    likeButton.classList.add('liked');
                    likeText.textContent = 'Вам нравится';
                    likeMessage.style.display = 'block';
                } else {
                    likeButton.classList.remove('liked');
                    likeText.textContent = 'Нравится';
                    likeMessage.style.display = 'none';
                }
            }
            
            likeButton.addEventListener('click', function() {
                if (isLiked === 'true') {
                    likes--;
                    isLiked = 'false';
                } else {
                    likes++;
                    isLiked = 'true';
                }
                
                localStorage.setItem('articleLikes', likes);
                localStorage.setItem('isLiked', isLiked);
                updateLikeButton();
                
                likeButton.style.transform = 'scale(0.95)';
                setTimeout(() => {
                    likeButton.style.transform = '';
                }, 200);
            });
            
            function isLocalStorageSupported() {
                try {
                    const testKey = '__test__';
                    localStorage.setItem(testKey, testKey);
                    localStorage.removeItem(testKey);
                    return true;
                } catch (e) {
                    console.error('LocalStorage не поддерживается:', e);
                    return false;
                }
            }
            
            if (!isLocalStorageSupported()) {
                likeButton.disabled = true;
                likeText.textContent = 'Лайки недоступны';
                likeCount.style.display = 'none';
            }
        });
